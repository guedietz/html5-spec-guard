# Campaign Manager 360 (CM360) — HTML5 Banner Specifications

Reference for automated compliance checks. Each check has a unique ID for traceability in reports.

---

## Package Requirements

### CM360-PKG-01: Archive Format
- **Allowed formats:** `.zip`, `.adz`
- **Severity:** FAIL

### CM360-PKG-02: Maximum File Count
- **Limit:** 100 files per archive
- **Severity:** FAIL

### CM360-PKG-03: Maximum Package Size
- **Limit:** 10 MB (10,485,760 bytes)
- **Severity:** FAIL

### CM360-PKG-04: Primary HTML File
- **Rule:** Archive must contain at least one `.html` or `.htm` file at the root level
- **Severity:** FAIL

---

## File Requirements

### CM360-FILE-01: Allowed File Types
- **Allowed extensions:** `.html`, `.htm`, `.js`, `.css`, `.jpg`, `.jpeg`, `.gif`, `.png`, `.json`, `.xml`, `.svg`, `.eot`, `.otf`, `.ttf`, `.woff`, `.woff2`
- **Severity:** FAIL
- **Note:** Files without extensions or with disallowed extensions must be flagged

### CM360-FILE-02: No Nested ZIP Archives
- **Rule:** Archive must not contain nested `.zip` or `.adz` files
- **Severity:** FAIL

---

## HTML Requirements

### CM360-HTML-01: Ad Size Meta Tag
- **Rule:** Primary HTML file must contain a `<meta>` tag declaring the ad dimensions
- **Pattern:** `<meta name="ad.size" content="width=X,height=Y">`
- **Validation:** `X` and `Y` must be positive integers
- **Severity:** FAIL

### CM360-HTML-02: Valid HTML Structure
- **Rule:** Primary HTML file must contain `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>` tags
- **Severity:** WARNING

## Click Tracking

### CM360-CLICK-01: clickTag Variable Present
- **Rule:** The JavaScript code must define a `clickTag` variable. Also detects multi-clickTag variants (`clickTag1`, `clickTag2`, etc.) and lists all exit variables found.
- **Patterns to search:**
  - `var clickTag` / `var clickTag1` / `var clickTag2` …
  - `let clickTag` / `let clickTag1` …
  - `const clickTag` / `const clickTag1` …
  - `window.clickTag` / `window.clickTag1` …
- **Severity:** FAIL

### CM360-CLICK-02: clickTag Not Hardcoded
- **Rule:** The `clickTag` value must not be a hardcoded URL. It should use URL parameters or be left as a placeholder.
- **Patterns that FAIL:**
  - `clickTag = "http://..."`
  - `clickTag = 'http://...'`
  - `clickTag = "https://..."`
  - `clickTag = 'https://...'`
- **Acceptable patterns:**
  - Assignment from URL parameters (e.g., `clickTag = getParameterByName(...)`)
  - Assignment from `window.location` parsing
  - Placeholder values like `""` or `''`
- **Severity:** FAIL

### CM360-CLICK-03: clickTag Readable (Not Minified)
- **Rule:** The clickTag implementation should be in unminified, human-readable code
- **Heuristic:** If the JS file containing clickTag has an average line length > 500 characters, flag as potentially minified
- **Severity:** WARNING

## Security

### CM360-SEC-01: HTTPS Only
- **Rule:** All URLs referenced in HTML, JS, and CSS files must use `https://` protocol
- **Detection:** Search for `http://` URLs (excluding `http://www.w3.org` namespace references)
- **Severity:** FAIL

### CM360-SEC-02: No Local Storage
- **Rule:** Code must not use `localStorage` or `sessionStorage`
- **Patterns to search:** `localStorage`, `sessionStorage`
- **Severity:** FAIL

## Polite Loading

### CM360-POLITE-01: Polite Load Threshold
- **Rule:** If total asset size exceeds 150 KB (153,600 bytes), the banner should implement polite loading
- **Detection:** Check for polite load patterns:
  - `window.addEventListener("load", ...)`
  - `document.addEventListener("DOMContentLoaded", ...)`
  - `Enabler.isInitialized()` or `Enabler.isPageLoaded()`
- **Enhanced message:** Reports total size, estimated initial payload (from `<head>` and pre-polite `<body>` references matched to ZIP entries), and polite load detection status
- **Severity:** WARNING
- **Note:** This is a heuristic check — cannot guarantee correctness

### CM360-POLITE-02: Initial Load Image Size
- **Rule:** If polite loading is used, the initial loading image should be under 40 KB (40,960 bytes)
- **Detection:** Look for images referenced before the polite load trigger
- **Severity:** WARNING
- **Note:** Heuristic — may not accurately identify the initial load image
